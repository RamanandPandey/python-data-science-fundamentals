# Python Data Science Fundamentals

A hands-on notebook covering core Python libraries used in Data Science — 
from data structures to real-world data analysis.

## Topics Covered
- Python Basics: Lists, Dicts, List Comprehension, Functions
- NumPy: Array operations, reshaping, broadcasting, statistics
- Pandas: DataFrames, filtering, groupby, handling missing values
- Mini Project: Student performance analysis — cleaning, EDA, CSV pipeline

## Tech Stack
Python · NumPy · Pandas

## How to Run
Open the notebook in Jupyter or Google Colab.
